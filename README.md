> Interface de e-commerce **premium**, focada em **conversão**, **performance** e **experiência mobile-first**, com estética inspirada no ecossistema Apple.

---

## ✨ Highlights

- 🎯 Design **High-End** com Glassmorphism e Bento Grid  
- ⚡ Micro-interações fluidas e animações com Framer Motion  
- 📱 Experiência **mobile-first** (feels like native app)  
- 🛒 Carrinho lateral persistente com feedback em tempo real  
- 🎨 Tipografia forte + layouts expansivos  

---

## 🚀 Stack Tecnológica

- **Next.js 14/15** — Renderização híbrida e performance
- **Tailwind CSS** — Estilização moderna e escalável
- **Framer Motion** — Animações e transições avançadas
- **Zustand** — Estado global simples e eficiente
- **Lucide React** — Ícones minimalistas

---

## 🛠️ Features

- ✔️ Layout totalmente responsivo  
- ✔️ Carrinho com persistência local  
- ✔️ Filtro dinâmico de categorias  
- ✔️ Hero com efeito Parallax  
- ✔️ Notificações de interação (UX feedback)  
- ✔️ Seções institucionais (manifesto, depoimentos, diferenciais)

## 🔧 Setup Local

### 1. Clone o repositório
```bash
git clone https://github.com/sGabDev/e-commerce-UI.git
```

### 2. Instale as dependências
```bash
npm install
```

### 3. Rode o projeto
```bash
npm run dev
```

### 4. Acesse
```
http://localhost:3000
```

---

## 📁 Estrutura

```text
src/
├── app/          # Rotas e layouts (App Router)
├── components/   # Componentes reutilizáveis
├── data/         # Mock de dados
├── store/        # Estado global (Zustand)
└── types/        # Tipagens
```

---

## 🔱 Conceito

A **PoseiShop** vai além de um e-commerce:  
é um experimento de **UX + engenharia front-end**, onde performance e estética coexistem.

Inspirado em marcas de luxo e no design system da Apple, o projeto demonstra como interfaces podem ser **imersivas, rápidas e emocionalmente envolventes**.
